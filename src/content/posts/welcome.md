这是我的个人博客。它不是一个复杂的网站，更像是一个长期记录本：把折腾过的网站、用过的工具、遇到的问题和解决办法都写下来。

## 这个网站用来做什么

目前主要记录这些内容：

- 个人网站搭建和部署过程
- GitHub、VS Code、EdgeOne Pages 等工具使用
- 本地修改、预览、提交、上线的完整流程
- 学习笔记、操作经验和一些个人想法

我希望每篇文章都尽量写得清楚一点，少一点术语，多一点实际步骤。

## 为什么要做成中文

这个网站主要给国内用户访问，所以页面、导航、按钮和文章内容都改成中文。

以后别人打开网站，不需要先理解英文模板，也不用猜每个按钮是什么意思。

## 后面怎么更新

博客内容保存在 GitHub 仓库里。本地改好文章后，提交并推送到 GitHub，腾讯 EdgeOne Pages 会自动重新部署。

简单说就是：

```txt
本地改文章 -> 提交到 GitHub -> 网站自动更新
```

后面会继续补充更多实用内容。
